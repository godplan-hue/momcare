# Project Summary
This project is a comprehensive web application designed to enhance user engagement through personalized content and settings management. It features a robust payment and subscription system utilizing the Polar API for PRO plans, allowing users to manage subscriptions seamlessly. Recent updates have focused on improving user login experiences, introducing a members management feature, and enhancing the pricing structure with a 30% discount on the PRO Yearly plan. The application now addresses user feedback regarding the login process and user interface, alongside improvements to the mobile menu design for better usability.

# Project Module Description
- **Authentication Module**: Manages user login/logout and session management using Atoms Cloud Auth, now featuring a customizable login dialog with separate login and signup buttons.
- **Profile Management**: Enables users to view and modify their profile information.
- **Feature Routing**: Facilitates navigation to various features within the application.
- **Payment Module**: Implements a subscription system with Free and PRO plans, including payment processing via Polar API.
- **Members Management**: Allows for auto-registration of members upon login and provides an admin view for managing members.

# File Description Inventory
- **backend/**: Contains all server-side code, including routes, models, and services.
- **frontend/**: Contains the client-side application code, including components and pages.
- **uploads/**: Directory for storing uploaded images and files.
- **i18n/**: Contains internationalization files for language support.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Frontend**: React, TypeScript, Tailwind CSS
- **Authentication**: Atoms Cloud Auth
- **Payment Processing**: Polar API
- **State Management**: React Context API

# Usage
1. **Install Dependencies**:
   - Navigate to the backend directory and run the dependency installation command.
   - Navigate to the frontend directory and run the dependency installation command.

2. **Build**:
   - Build the backend and frontend applications using their respective build commands.

3. **Run**:
   - Start the backend server and the frontend development environment using the appropriate commands.
